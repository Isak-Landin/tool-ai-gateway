# Project Dev

_Idea development tracking across all project modules. This is not a codebase representation — it tracks ideas, intent, and cross-module relationships as they develop._

Each module page is divided into three sections:

- **Confirmed** — ideas and behavior that are established and agreed upon
- **Being Developed** — ideas actively being mapped or iterated
- **Uncertain** — ideas that are noted but not yet resolved

Within each section, ideas are grouped by the other modules they touch.
